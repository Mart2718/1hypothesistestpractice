Problem 1 of 5

Quick-reference summary of both hypothesis tests — formulas, conditions, the 4-step process, and tail directions.

1-Proportion Z test
Use for: categorical data (yes/no), testing p
z = (p̂ − p₀) / √(p₀q₀ / n)
  • Independent: n ≤ 5% of population
  • Random sample
  • np₀ ≥ 10 AND nq₀ ≥ 10
1-Sample T test
Use for: quantitative data, testing μ (σ unknown)
t = (x̄ − μ₀) / (s / √n)
  • Independent: n ≤ 5% of population
  • Random sample
  • n > 30 OR approximately normal population
The 4-step process (both tests)
Step 1
State parameter & hypotheses
Define μ or p in context. H₀ uses =. Hₐ uses <, >, or ≠.
Step 2
Name the test
1-Proportion Z Test or 1-Sample T Test. State α.
Step 3
Calculate & find p-value
Compute the test statistic. Tail from Hₐ.
Step 4
Write the conclusion
Compare p-value to α. Reject or fail to reject. State in context.
Tail direction — reading Hₐ
Hₐ: param < value
Left-tailed · use lower tail p-value
Hₐ: param > value
Right-tailed · use upper tail p-value
Hₐ: param ≠ value
Two-tailed · double the one-tail p-value
